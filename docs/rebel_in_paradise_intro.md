# TickPay - 基于 Monad 和 EIP-7702 的无感流式微支付协议

**参赛黑客松**：Rebel in Paradise AI 黑客松
**参赛赛道**：Agent-native Payments
**关键词**：微支付、自动化结算、EIP-7702、账户抽象、流式支付

## 项目概述

TickPay 是一个构建在 Monad 高性能公链上的**流式微支付与自动化结算系统**，专为 AI Agent 经济和高频消费场景（如视频按秒计费、AI API 按次调用等）设计。

通过深度整合 EIP-7702 账户抽象协议，TickPay 允许用户将自己的 EOA（外部拥有账户）临时委托给计费智能合约，授权服务端的 Relayer（代理/智能体）在用户消费服务（如观看视频、调用大模型）期间自动进行**高频、小额的代币扣款**。服务结束后，授权可被瞬间撤销，实现真正的“即插即用”与“用多少付多少”。

## 为什么契合 Agent-native Payments 赛道？

TickPay 完美契合了赛道要求的**微支付、订阅制、自动化结算**以及**智能体间的价值流转**：

1. **极致的微支付体验 (Micro-payments)**：依托 Monad 极高的 TPS 和低延迟特性，TickPay 实现了真正的“按秒计费”。例如在视频场景中，每 10 秒自动结算一次费用，彻底打破传统包月订阅的壁垒，让微小价值得以流畅转移。
2. **自动化无感结算 (Automated Settlement)**：用户仅需在开始服务时签署一次 EIP-712 授权，后续的连续扣费全部由后台的 Keeper / Agent 自动完成，它会定期发起 `charge()` 上链结算，全程无需用户二次干预。
3. **Agent 驱动的支付流转**：系统中的 Relayer 本质上扮演了一个自动化结算 Agent 的角色。它根据用户（或其他 Agent）实际消耗的资源（时间、算力等），自主向区块链提交符合 EIP-7702 标准的 Type 4 交易，驱动智能体间的价值流转。

## 核心技术与创新点

### 1. 创新的 EIP-7702 授权模式

传统的 ERC-20 授权 (Approve) 容易带来无限授权的安全隐患，且无法控制具体的扣款逻辑。TickPay 采用 EIP-7702 机制：

- 用户只需签署 EIP-712 签名，将 EOA 的行为逻辑临时委托给特定的计费逻辑合约。
- 这种委托是受限且带有生命周期的，支持配置 `maxCost`（最高扣款上限）和 `maxSeconds`。
- 支持在停止服务时立刻发送委托置空的交易（`delegate=0x0`），实现硬核级别的权限即时回收。

### 2. Keeper 限定的安全机制

为了防止恶意扣款，TickPay 智能合约在业务逻辑上限制了只有特定的授权 Agent（Keeper）可以执行计费操作。系统加入了基于 Nonce 的防重放保护，并严格审计时间戳流逝情况，确保资金的绝对安全。

### 3. 原生代币安全隔离

考虑到 Monad 生态中 EIP-7702 代理账户需要预留 10 MON 的底层限制，TickPay 巧妙地规避了原生代币转移。所有流式支付均通过指定的 ERC20 资产完成结算，消除了消耗原生 Gas 代币储备的风险，也为未来的稳定币结算打下基础。

## 应用场景与生态扩展 (Agent-to-Agent)

为了证明底层架构的极强通用性并方便开发者快速接入，我们已经**封装了完整的 TickPay SDK**（包含客户端组件与服务端 Session 引擎），并在代码库中提供了一个 **Agent-to-Agent (智能体对智能体) 的最小化示例 (Minimal Relayer)**。

通过这套开箱即用的基础设施，TickPay 可被无缝扩展到各种 Agent-native 场景：

- **AI Agent API 实时计费 (已具备底层支持)**：按实际生成的 Token 数量或计算时间，由一个消费端 Agent 向另一个提供服务的 Agent 自动结算费用，完全基于代码逻辑运转，全程无需人工干预。
- **视频与内容流式付费**：如本项目的演示案例，基于真实观看时间或内容消耗进度进行秒级无感扣费。
- **去中心化云算力/带宽租赁**：按秒级或毫秒级使用量自动化支付 GPU 算力集群、去中心化物理基础设施 (DePIN) 的使用费用。
- **高频游戏经济系统**：玩家或游戏内的 NPC 智能体无需频繁唤起钱包签名，即可在游戏进程中实现资产或道具的高频流转。

基于 TickPay SDK，任何 AI Agent 都可以通过几行代码迅速装备上“流式收款”与“自动化支付”的经济组件。

## 总结

TickPay 结合了 Monad 的高性能底座与 EIP-7702 的前沿标准，为“智能体经济”和“Agent-native Payments”赛道提供了一套完整的、生产级别的**自动化微支付基础设施**，让价值的流转像信息流一样顺滑。
